What is MySnipItTool?

MySnipItTool is a screenshot utility that came about from frustrations of Windows' built-in screenshot tools, which only provide limited markup tools and require saving screenshots (or copying them somewhere) before taking new ones. Sometimes, you need to take multiple screenshots fairly quickly, then edit them later, or you just don't have time to save them between.

Features

Multitabular Workflow

Multiple screenshots can be taken and edited in separate tabs.

Annotation tools

Multiple tools available, including free draw, line, text and shape tools.

Fast and Lightweight

MySnipItTool is very lightweight, requiring little memory and disk space.

Open Source

Completely free and open source for those who want to do more than just use the tool!
